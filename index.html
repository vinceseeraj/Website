Abstract

The goal of this project was to create a system that could recognize speech patterns and distinguish between speakers. To do this, we needed to follow the distinct voice recognition matching techniques. This involved the use of the MFCC, or Mel-Frequency Cepstrum Coefficients, algorithm, and VQ, or Vector Quantization method. The MFCC uses DSP methods of Hamming window, Fourier Transforms, and Frame Blocking, to achieve the extraction of voice. In turn, these are implemented into the VQ method for the system to recognize and search for a pattern from the Mel-Frequency, so the system can accurately detect the correct voice.
